Capstone Demo

A suite of demo applications hosted in the Amazon cloud and configured for the handling of Service Plans.

For further details please contact: Richard Shaw, +44 (0)7748 630883

Preparation

Before running the demo you will need to:

  • Start the Amazon EC2 instances here
    (If you don't know which ones to start, start them all! Instances are set to automatically shutdown, if ticked, at 7pm each night.)
  • Wait a few minutes for the domain names to re-point to the new instances
  • Clear down the data (optional) by logging into the Users' Portal here as a super-user, going to the Scheduler, and triggering the 'Clear Down' job.
